◆ EventType

enum UndertaleModLib.Models.EventType : uint

The types an UndertaleGameObject.Event from an UndertaleGameObject can be.

Note, that subtypes exist as well.

Enumerator
Create 

A creation event type. Has no subtypes, it's always 0.

Destroy 

A destroy event type. Has no subtypes, it's always 0.

Alarm 

An alarm event type. The subtype is 0-11, depending on the alarm id.

Step 

A step event type. The subtype is EventSubtypeStep.

Collision 

A collision event type. The subtype is the other UndertaleGameObject's id.

Keyboard
