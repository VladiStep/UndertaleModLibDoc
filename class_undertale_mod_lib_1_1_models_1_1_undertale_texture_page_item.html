Detailed Description

A texture page item in a data file.

The way a texture page item works is:
It renders in a box of size BoundingWidth x BoundingHeight at some position.
TargetX, TargetY, TargetWidth and TargetHeight are relative to the bounding box, anything outside of that is just transparent.
SourceX, SourceY, SourceWidth and SourceHeight are part of the texture page which are drawn over TargetX, TargetY, TargetWidth, TargetHeight.
