Detailed Description

An object representing a Game Maker: Studio data file.

This is basically the heart of the data file, which is usually named data.win, data.unx, data.ios or data.droid, depending for which OS the game was compiled for..
It includes all the data within it accessible by either the FORM-Chunk attribute, but also via already organized attributes such as Backgrounds or GameObjects. TODO: add more documentation about how a data file works at one point.
