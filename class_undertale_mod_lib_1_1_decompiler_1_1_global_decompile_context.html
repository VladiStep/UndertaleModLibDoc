 A cache of function to actual name mapping. GMS2.3+ sometimes (usually when dealing with global scripts) calls method functions using the legacy call operator, passing the anonymous function directly. This dictionary contains a map from UndertaleFunction to its actual name, obtained by decompiling the parent CodeObject and looking for the assignment to global variable with function name.
 

Detailed Description

The DecompileContext is global for the entire decompilation run, or possibly multiple runs. It caches the decompilation results which don't change often to speedup decompilation.
