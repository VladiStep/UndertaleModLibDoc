void UndertaleModLib.Scripting.IScriptInterface.ImportASMFile ( string  fileName,
bool  doParse = true,
bool  nukeProfile = true,
bool  checkDecompiler = false,
bool  throwOnError = false 
)

Replaces/Imports all GM-Bytecode ASM in a specific code entry with a specified file.

Parameters
fileNameThe path to the ASM file. This file needs to A) contain valid GM-Bytecode ASM and B) the filename needs to be the name of the code entry that shall get replaced
If the entry does not exist, it will be created!
doParseWhether the code entry should get linked. In other words, have special handling for scripts, globals and object code.
nukeProfileWhether or not to nuke the profile entry for profile mode.
checkDecompilerIf this is false an empty string will be used for replacing the code entry in the case that anything fails.
throwOnErrorWhether a ScriptException will be thrown on any errors.
