Detailed Description

The exception that is thrown when trivial errors happen during runtime of UndertaleModTool scripts.
This exception does not contain a stacktrace and should more be handled like an error message that stops execution of the currently running script.

if (Data is null) throw new ScriptException("Please load data.win first!);
